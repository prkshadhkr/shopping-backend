## How to use

### Developers
  1. Clone the project
  2. Create **secret.js** in root directory.
  3. In secret.js add following:<br>
     _const STRIPE_SECRET_KEY = YOUR_STRIPE_KEY <br>
     export { STRIPE_SECRET_KEY }_ <br>
     (Note: visit https://stripe.com/ to get YOUR_STRIPE_KEY)<br>
